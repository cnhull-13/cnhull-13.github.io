AJAX Application

Explanation of AJAX

AJAX is simply the combination of JavaScript and XML; simply put, XML stores data, and we use JavaScript to access that data and present it to the user. JavaScript is able to open the XML file, and read the contents. In this case, the JavaScript then constructs additional HTML to be inserted, and then inserts the parsed data at the appropriate elements.

REST API

Pokeapi offers a robust, read-only API for other sites to access thousands of data entries from the Pokemon series of Video Games. While it was originally started by one person, the project has grown and evolved, and is now maintained by a collective of Pokemon fans. Pokeapi open access--no authentication required--and completely free for all users, though they do request that data be cached locally, where possible. While rate limiting was previously necessary, they were able to remove those rate limits after migrating all of the data to a static JSON format.
