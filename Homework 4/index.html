Progressive Web Apps have a variety of benefits and features, including, but not limited to:

  • No platform specific development tools required: you can develop a PWA with standard web development tools
  • PWAs can be installed straight from a webpage--no stores required!
  • Installed PWAs work very much like any other kind of app, and can even be used offline (if the developers have provided support for that)
  • PWAs can continue to run in the background, pulling and pushing information, and displaying notifications
  • PWAs are a great midpoint between a website and a true application, providing many benefits of both!
