Cordova

  • Created in ~2009
  • Formerly known as PhoneGap
  • Originally created by Nitobi; Adobe later bought the studio and released the code to the Apache Software Foundation
  • In a nutshell, bundles up web applications into a mobile-usable form. While not the most efficient, this means basic web development experience is enough to put something together, and the same application can be used across multiple platforms.

React Native

  • Created by Facebook (now Meta) in ~2015
  • Supports JavaScript, Java, C++, Python
  • A near-native experience; as best I understand, React Native uses JavaScript to implement Native elements.
  • fairly easy to get running
  • installation requires node.js, but plenty of other things also require node.js, so that's not a huge ask
  • the lecture guide was in unix, which isn't my preferred way of doing things
  • unable to open link from windows to iphone. it just says there's no data to be read, or just acts like a prompt to search

Flutter

Cordova

Cordova primarily works in JavaScript, HTML, and CSS, making it an easy transition for people who already have some web development experience.

ReactNative

React Native also works mostly in JavaScript, giving it many of the same benefits as Cordova. However, it does not use HTML and CSS to handle its layout.

Flutter

Flutter, developed by Google, uses a language called Dart, also developed by Google. It uses a system of "widgets" to create applications.

Cordova

Because Cordova works primarily in JavaScript, it also has access to JavaScript's large library of third-party plugins and other add-ons.

ReactNative

Just like Cordova, React Native also has access to JavaScript's large library of third-party plugins and other add-ons.

Flutter

Flutter's library is more limited, with most add-ons coming directly from Google

Cordova

Because Cordova effectively packages a web app in a mobile wrapper, Cordova's performance may suffer with large, complicated applications. Most applications won't be affected.

ReactNative

React Native has fairly high performance, outpacing Cordova, but falling short of Flutter's. Again, the difference won't be apparent on most applications.

Flutter

Flutter supposedly peforms the closest to a fully native app, and is better able to handle large, complex apps that Cordova and React Native may struggle with.
